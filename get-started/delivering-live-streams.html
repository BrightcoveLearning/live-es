Entrega de transmisiones en vivo

Este tema describe las opciones que ofrece Brightcove para transmitir transmisiones en vivo.

La función de transmisión de video en vivo de Video Cloud le permite entregar transmisiones de transmisión en vivo de alta calidad a cualquier dispositivo o navegador. Puede entregar transmisiones en vivo sin ningún desarrollo personalizado o integración compleja; sin embargo, las API están disponibles si tiene necesidades personalizadas o requiere integraciones personalizadas.

Nota: La compatibilidad con transmisión en directo solo está disponible para los editores de Video Cloud Enterprise.

Video Cloud también admite transmisión en vivo con DVR. Cuando se configura un reproductor de Brightcove con el complemento DVRUX, las transmisiones en directo actúan como si se estuvieran reproduciendo a través de la grabadora de vídeo digital del espectador.

Brightcove ofrece varias opciones para transmitir transmisiones en vivo. Este tema contiene las siguientes secciones:

  • Codificación de hardware frente a la basada en la nube
  • Transmisión en vivo simple usando un codificador basado en software
  • Transmisión en vivo usando el módulo Video Cloud Live
  • Transmisión en vivo usando un codificador basado en hardware

Codificación de hardware frente a la basada en la nube

Brightcove admite codificación basada en hardware y en la nube. Varios factores pueden determinar si la codificación basada en hardware o en la nube es la mejor para un evento.

Cuándo usar la codificación por hardware

  • Alimentaciones lineales 24x7 donde se conoce la capacidad
  • Una arquitectura basada en una única ubicación
  • Soporte de formato de transmisión

Cuándo usar la codificación en la nube

  • Para eventos
  • Cuando se desconocen los requisitos de capacidad
  • Para organizaciones descentralizadas
  • Cuando se desconoce el tiempo del evento y la velocidad es importante
  • Si hay cuellos de botella en el ancho de banda

Transmisión en vivo simple usando un codificador basado en software

Esta opción le permite utilizar Akamai Stream Packaging y un codificador basado en software para transmitir un evento en vivo a un Brightcove Player. Para más información, ver Creación de una transmisión en vivo con Akamai Stream Packaging y Telestream Wirecast.

Transmisión en vivo usando el módulo Video Cloud Live

El módulo Video Cloud Live proporciona una interfaz fácil de usar para ayudarlo a transmitir un evento en vivo a dispositivos móviles y de escritorio. No se requiere programación. Después de ingresar la información de su evento en el módulo Live, se le proporcionará un conjunto de configuraciones de codificador que conectará a su codificador en el sitio. Toda la transcodificación se realiza en la nube, por lo que no necesita un transcodificador basado en hardware. Para obtener más información, consulte uno de estos documentos.

API en vivo

Live API es una API basada en REST que le permite crear y administrar eventos de transmisión en vivo. Las características opcionales incluyen:

  • Inserción de anuncios del lado del servidor (SSAI)
  • Cifrado AES-128
  • Cree activos de video a pedido a partir de clips tomados de la transmisión en vivo
  • Capacidad DVR
  • Varias CDN

Para obtener más información, consulte el Documentación de API en vivo.

Transmisión en vivo usando un codificador basado en hardware

Esta opción utiliza un codificador basado en hardware para codificar y entregar transmisiones en vivo a su CDN. En el lado de Video Cloud, crea activos remotos para alojar las transmisiones y luego puede publicar un reproductor para su transmisión en vivo.
