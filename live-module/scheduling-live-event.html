--- title: Programación de eventos SEP en vivo parent: Live Module ---

Inicio / parada automática de eventos en vivo con puntos de entrada estáticos

Esta función permite al usuario programar y administrar la hora de inicio y finalización de uno o varios eventos de Live Job.

Prerrequisitos

Para programar eventos, se debe crear un tipo de evento en vivo SEP (punto de entrada estático).

Actualmente, no se admite la programación de grupos redundantes.

Recomendaciones y mejores prácticas

Para obtener los mejores resultados, siga las instrucciones a continuación.

  1. Se pueden programar seis activaciones pendientes a la vez. Si necesita más, comuníquese con el servicio de atención al cliente.
  2. Una vez que se activa el SEP, el tiempo máximo de espera predeterminado sigue siendo válido (consulte API en vivo: Puntos de entrada estáticos para obtener más información).
  3. Tenga en cuenta que las limitaciones descritas en el Puntos de entrada estáticos siguen siendo válidos.

Panel de programación en la sala de control

Si una cuenta en vivo está habilitada para Auto Start / Stop, un panel de programación estará disponible en la esquina superior derecha de la sala de control, debajo del reproductor de vista previa.

Para utilizar Auto Start / Stop, siga las instrucciones a continuación:

  1. En el módulo En vivo, cree un nuevo evento en vivo con un punto de entrada estático o visite uno creado previamente.
  2. En la Sala de control, estará disponible un nuevo panel denominado Scheduling. Hacer clic + Agregar horario para crear una nueva activación y desactivación automática.
  3. Modifique las fechas de inicio y finalización a la activación y desactivación deseadas, respectivamente.
    Si bien no impedimos que se ejecuten varias programaciones al mismo tiempo, las filas en conflicto se mostrarán con una advertencia. Es probable que los horarios en conflicto fallen si no se abordan antes de la hora de activación. Para evitar la advertencia, basta con eliminar una de las filas o ajustar las horas.
  4. Para crear un segundo horario, vuelva a hacer clic en el + Agregar horario botón. La + Agregar horario El botón se desactivará automáticamente cuando se alcance el límite de activaciones pendientes.
  5. Para eliminar una programación, haga clic en el icono.
  • Los eventos de programación se volverán amarillos como advertencia cuando haya horarios en conflicto.
  • Los eventos programados se volverán rojos cuando haya una falla en la transmisión por cualquier motivo.
  • Los horarios fallidos permanecerán visibles durante 7 días.

Visualización de próximas activaciones

  1. Una vez creadas las activaciones programadas, puede verlas en el Gestionar eventos> En espera pestaña. El próximo próximo Comienzo y Parada las fechas se muestran debajo de Próximamente título.
  2. La Show El cuadro de selección le permite mostrar solo los eventos programados.
  3. Una vez que comience un evento, se mostrará en el Vivir pestaña, y la Comienzo el tiempo se desactivará.
