Parada de inicio de empleo

Con el punto final JobStartStop, puede programar la activación y desactivación de un trabajo en vivo existente. A la hora especificada, nuestro sistema realizará las llamadas API necesarias para activar y desactivar un trabajo en su nombre.

Requisitos

  • La clave de API en vivo para su cuenta.
  • Su cuenta real debe tener habilitada la anulación del programador.
  • Un trabajo en vivo existente con "estático": verdadero.

Crear un flujo de trabajo JobStartStop

Digamos que queremos programar un evento de 2 horas que comenzará a las 2021-02-02T14: 00: 00 + 00: 00.

Las marcas de tiempo deben estar en UTC formateadas como marca de tiempo Unix en segundos. Puedes usar https://www.unixtimestamp.com/ para la conversión.

Tenga en cuenta que las limitaciones de tiempo de activación descritas aquí todavía se aplican a las activaciones programadas, por lo que recomendamos programar una hora que sea al menos 60 segundos antes de la hora de inicio deseada.

Para este ejemplo, elegiremos 5 minutos antes de la hora de inicio prevista.

comienzo: 2021-02-02T13: 55: 00 + 00: 00 = 1612274100 parada: 2021-02-02T 16:00:00 + 00:00 = 1612281600

Para crear el flujo de trabajo, realice la siguiente solicitud

POST https://api.bcovlive.io/v1/scheduler/jobstartstop Encabezados: Tipo de contenido: application/json X-API-KEY: cuerpo clave de la API: { "job_id": "your Job ID", "activation": { "time_utc": 1612274100 }, «desactivación»: { "time_utc": 1612281600 }, «descripción»: «Mi emisión de martes»}

Debería recibir una respuesta similar a

{ "account_id": "your Account ID", "activation": { "state": "pending", "time_utc": 1612274100 }, «desactivación»: { "state": "pending", "time_utc": 1612281600 }, «descripción»: «Mi emisión del martes», «job_id»: «su ID de trabajo», «tipo»: «jobstartstop», «workflow_id»: «d44cf29f4f184757ac0995beb8a0097e»}

Actualizar un flujo de trabajo JobStartStop

Puede actualizar los tiempos de activación y desactivación de un flujo de trabajo JobStartStop. Estos solo se pueden actualizar si "estado": "pendiente".

Siguiendo el ejemplo anterior, queremos detener el evento 30 minutos antes de lo programado.

PONGA https://api.bcovlive.io/v1/scheduler/jobstartstop/workflow_id Encabezados: Tipo de contenido: application/json X-API-KEY: cuerpo clave de la API: { "deactivation": { "time_utc": 1612279800 }}

Recibirá el flujo de trabajo actualizado en la respuesta.

{ "account_id": "your Account ID", "activation": { "state": "pending", "time_utc": 1612274100 }, «desactivación»: { "state": "pending", "time_utc": 1612279800 }, «descripción»: «Mi emisión del martes», «job_id»: «su ID de trabajo», «tipo»: «jobstartstop», «workflow_id»: «d44cf29f4f184757ac0995beb8a0097e»}

Cancelar un flujo de trabajo JobStartStop

Puede cancelar la activación y desactivación con un ELIMINAR solicitud. Puede hacer esto después de la activación para cancelar la desactivación.

BORRAR https://api.bcovlive.io/v1/scheduler/jobstartstop/workflow_id Encabezados: X-API-KEY: su clave API

Recibirá el flujo de trabajo actualizado en la respuesta.

{ "account_id": "your Account ID", "activation": { "state": "cancelled", "time_utc": 1612274100 }, «desactivación»: { "state": "cancelled", "time_utc": 1612279800 }, «descripción»: «Mi emisión del martes», «job_id»: «su ID de trabajo», «tipo»: «jobstartstop», «workflow_id»: «d44cf29f4f184757ac0995beb8a0097e»}

Notificaciones

Opcionalmente, puede configurar notificaciones al crear el flujo de trabajo. Debe proporcionar una URL para que nuestro servicio CORREO para. Enviaremos una notificación cuando el programador realice las llamadas de activación y desactivación. También puede configurar para recibir una notificación norte segundos antes de la activación y / o desactivación. Ver el Guía de notificaciones para más detalles.

POST https://api.bcovlive.io/v1/scheduler/jobstartstop Encabezados: Tipo de contenido: application/json X-API-KEY: cuerpo clave de la API: { "job_id": "your Job ID", "activation": { "time_utc": 1612274100, "notification": 600 }, «desactivación»: { "time_utc": 1612281600, "notification": 600 }, «descripción»: «Mi emisión de martes», «notification_url»: "http://example.com/live/jobstartstop/callbacks"}
