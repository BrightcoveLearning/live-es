    Administrar la configuración de Live

    En este tema aprenderá a administrar la configuración de Brightcove Live.

    La configuración Live le permite administrar las pizarras de relleno y las credenciales Live.

    Nota: Las configuraciones de anuncios del lado del servidor utilizadas por Brightcove Live se configuran como parte de la administración Configuración de anuncios del lado del servidor. Para obtener más información, consulte Configuración de la configuración de anuncios del lado del servidor.

    Administración de pizarras de relleno

    Las pizarras de relleno son contenido que se puede publicar cuando hay huecos en un salto de anuncio de transmisión en vivo. Las pizarras se pueden usar para proporcionar un mensaje «Be right back» o cualquier otro contenido que desee mostrar. Todos los formatos de entrada de Zencoder son compatibles.

    Nota: Las pizarras de relleno también se pueden administrar usando la API Live. Para obtener más información, consulte Brightcove Live API: Inserción de anuncios en el lado del servidor (SSAI).

    Para agregar una pizarra de relleno, siga estos pasos.

    1. Haga clic en el enlace Administrador > Live Settings en el encabezado de Studio.
      Nota: Debes ser administrador de cuentas para acceder al menú ADMIN.
    2. Haga clic en + Añadir pizarra de relleno.
    3. Introduzca un nombre.
    4. Introduzca la URL de origen de la pizarra.
    5. Haga clic en Guardar. Comenzará el proceso de ingestión de la pizarra.

    Para eliminar una pizarra de relleno, haga clic en el botón Eliminar () situado junto a la pizarra de relleno.

    Administrar credenciales

    Tanto el Live API como el módulo Live utilizan las credenciales. Si va a crear una nueva credencial de Live Video Cloud, primero debe crear una credencial de autenticación API. El ID de cliente y el secreto de cliente que se crean se utilizarán al crear la credencial de Live Video Cloud. Al crear credenciales FTP, S3 o RTMP, no se necesita una credencial de autenticación API.

    Si va a crear una credencial S3, FTP o RTMP, vaya al paso 5. Para crear una credencial de Live Video Cloud, comience creando una credencial de API.

    1. Haga clic en el enlace ADMIN del encabezado de Studio.
      Nota: Debes ser administrador de cuentas para acceder al menú ADMIN.
    2. Haga clic en el enlace Autenticación vía API. Se abrirá la página Autenticación vía API, en la que aparecerán sus registros de cliente actuales.
    3. Haga clic en Registrar nueva aplicación para crear una nueva credencial API. Se deben seleccionar las siguientes API.
      Nota: Para obtener información sobre cómo crear credenciales de autenticación de API, vea Administrar credenciales de autenticación de API.
    4. Cuando se cree la credencial, guarde los valores de ID de cliente y secreto de cliente. Estas credenciales serán necesarias al crear la credencial de Live Video Cloud.
    5. Haga clic en el enlace Administrador > Live Settings en el encabezado de Studio.
    6. Haga clic en + Agregar credencial.
    7. Introduzca una etiqueta. Este es el nombre que se usará internamente para referirse a la credencial.
    8. Seleccione un tipo:
      • FTP - Se utiliza para guardar el activo VOD en una ubicación ftp (uso de API)
      • RTMP - Se utiliza para crear credenciales RTMP para ser utilizadas por el codificador
      • S3 : se utiliza para guardar el activo VOD en una cuenta de Amazon S3 (uso de API)
      • Video Cloud : se utiliza para guardar el activo VOD en una cuenta de Video Cloud (uso de API)
    9. Para Clave pública/Nombre de usuario, escriba lo siguiente según el tipo de credencial:
      • FTP - Introduzca el nombre de usuario para el sitio FTP
      • RTMP - Introduzca el nombre de usuario que usará el codificador para autenticar
      • S3 - Ingresa el nombre de usuario de la cuenta S3
      • Video Cloud : introduzca el ID de cliente generado para la credencial de autenticación de API
    10. Para Clave privada/Contraseña, escriba lo siguiente según el tipo de credencial:
      • FTP - Introduzca la contraseña para el sitio FTP
      • RTMP - Introduzca la contraseña que usará el codificador para autenticar
      • S3 - Introduzca la contraseña de la cuenta S3
      • Video Cloud : introduzca el secreto del cliente generado para la credencial de autenticación de API
    11. Haga clic en Guardar.

    La credencial aparecerá en la lista de credenciales. Para eliminar una credencial, haga clic en el botón Eliminar () situado junto a la credencial.
